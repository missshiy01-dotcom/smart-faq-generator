# Smart FAQ Generator

Course Project 1.2 - Information Management

## Features
- AI-powered FAQ generation using Google Gemini
- Multiple document formats (PDF, TXT, MD)
- Export to JSON, Markdown, HTML

## Quick Start

1. Install dependencies:
```bash
   pip install -r requirements.txt
```

2. Get free API key from: https://makersuite.google.com/app/apikey

3. Run:
```bash
   streamlit run app.py
```
